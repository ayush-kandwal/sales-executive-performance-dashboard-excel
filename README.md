# sales-executive-performance-dashboard-excel

## Overview
This project is an end-to-end Excel dashboard built to evaluate sales executive performance across regions by analyzing total sales, target achievement, and performance gaps.

## Business Problem
Sales managers need a quick and clear way to:
- Compare performance of sales executives
- Track target achievement
- Identify underperforming regions and individuals

## Dataset
- Sales data for individual sales executives
- Region-wise classification
- Daily sales figures
- Monthly sales targets

## Key Metrics
- Total Sales
- Target Hit %
- Away From Target %
- Executive-wise and Region-wise performance

## Tools & Techniques
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Slicers for interactivity
- Excel formulas for KPI calculations

## Dashboard Insights
- Identifies top-performing sales executives
- Highlights executives farthest from targets
- Compares regional performance trends

## Use Case
Useful for sales managers and business teams to monitor individual performance and take corrective actions based on data.
